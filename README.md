# WP Image Watermark

WordPress 圖片浮水印外掛，支援文字與圖像浮水印，可自動/手動/批次套用，並可隨時停用自動浮水印。

## 功能

- **自動浮水印開關** — 上傳時自動套用，可隨時啟用或停用
- **文字浮水印** — 自訂文字、字體大小、顏色、不透明度
- **圖像浮水印** — 上傳自訂圖片（建議透明 PNG）
- **9 點定位** + X/Y 偏移精確控制
- **批次處理** — 全部套用 / 全部還原
- **媒體庫整合** — 列表欄位顯示狀態、行動連結
- **影像保護** — 右鍵/拖放/DevTools 防護
- **雙引擎支援** — ImageMagick（優先）或 GD
- **備份機制** — 套用前自動備份，可完整還原

## 安裝

1. 上傳外掛到 `wp-content/plugins/`
2. 後台啟用
3. 前往「媒體 → 浮水印設定」

## 需求

- WordPress 5.8+
- PHP 7.4+
- GD 或 ImageMagick 擴充套件

為您的網站圖片自動或手動新增浮水印，支援文字與圖像浮水印，可批次處理並隨時停用自動功能。

== Description ==

WP Image Watermark 是一個功能完整的浮水印外掛，具備以下功能：

**浮水印類型**
* 文字浮水印 — 自訂字體大小、顏色、不透明度
* 圖像浮水印 — 上傳 PNG/SVG 作為浮水印圖章

**靈活應用**
* 上傳時自動套用（可隨時開啟/關閉）
* 媒體庫列表顯示浮水印狀態
* 單張手動套用或還原
* 批次套用/還原所有圖片

**進階自訂**
* 9 個位置選項 + X/Y 偏移精確控制
* 比例縮放（相對圖片寬度百分比）
* 不透明度 0-100%

**影像保護**
* 停用右鍵選單
* 停用拖放
* 開發者工具偵測（實驗性）

**技術**
* 支援 JPEG、PNG、WebP
* 自動選擇 ImageMagick（優先）或 GD 引擎
* 原始圖片備份，可完整還原

== Installation ==

1. 上傳 `wp-image-watermark` 資料夾到 `/wp-content/plugins/`
2. 在 WordPress 後台啟用外掛
3. 前往「媒體 → 浮水印設定」進行設定
